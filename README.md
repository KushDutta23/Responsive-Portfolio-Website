🌐 My First Webpage

This is a simple HTML + CSS project that demonstrates the basics of creating a webpage with styling. It is ideal for beginners starting with web development.

🚀 Features

📄 Basic HTML structure

🎨 Internal CSS styling

📱 Responsive design using viewport meta tag

🧾 Clean and minimal UI

🧠 Concepts Used

HTML5 structure (<!DOCTYPE html>, <head>, <body>)

Meta tags for responsiveness

Internal CSS (<style> tag)

Text styling and layout
